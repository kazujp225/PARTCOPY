事業の定義
一言で何を作るか

既存URLを入れると、日本企業サイトを semantic block（意味単位の再利用可能ブロック）に分解し、同業比較・改善提案・新サイト初稿生成までやる “Japan Site Genome OS”。

ここでの核心は、raw HTML を編集対象にしないこと。
編集対象は次の3層に分ける。

収集層
HTML、CSS、JS、スクリーンショット、DOM、ネットワーク情報

意味層
Hero、Feature、CTA、FAQ、Pricing、Footer、Contact などの semantic block

生成層
新しいテーマ、コピー、レイアウト、デザインシステムに基づく再構築コード

あなたが前から感じていた「ミラーしたものは修正しづらい」の正体はここだ。
ミラーは収集層で止まっている。商用価値が出るのは意味層に変換できた時だけ。

どこで勝つか
勝ち筋

汎用サイトビルダーではなく、既存サイトの“構造化と再構成”に寄せる。
